# DispatcherOne Desktop

## Stack

Frontend (UI)
- HTML / CSS / TypeScript (React)

Desktop wrapper
- Tauri (smaller, faster, more secure than Electron)

Backend
- Node.js or Python (FastAPI)

Database
- SQLite (local) -> PostgreSQL (later, if networked)

Platforms
- macOS + Windows 11

Software to install
- VS Code
- Node.js
- Python (optional)

## Why this works for a dispatcher system
- Cross-platform for macOS + Windows 11
- Easy forms, tables, dashboards
- Real-time updates for dispatch status, drivers, and jobs
- Scales from a local app to a networked system
- Huge ecosystem and tutorials
